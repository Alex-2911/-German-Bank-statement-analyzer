# Contributing

Thanks for contributing to German Bank PDF Statement Analyzer.

## Development flow

1. Fork and clone.
2. Run `npm install`.
3. Run `npm run dev`.
4. Keep changes modular and typed.
5. Add/update docs for user-visible behavior.

## Standards

- Local-first by default.
- No personal or private financial data in commits.
- No references to specific banks in labels, examples, or screenshots.
- Use synthetic demo files only.
